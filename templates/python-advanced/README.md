# Advanced Python Widget Template

Advanced Python template with rich visualizations, error handling, and dual-mode support.

## Features

- Dashboard vs Extended mode support
- Sparklines and graphs
- Statistical tables
- Error handling
- Easy to extend with Python libraries

## Usage

```bash
serverhub new-widget python-advanced --name api-monitor
```

## Template Variables

- **WIDGET_NAME** (required): Widget identifier
- **WIDGET_TITLE** (optional): Display title
- **REFRESH_INTERVAL** (optional): Refresh interval in seconds
- **AUTHOR** (optional): Widget author name
- **DESCRIPTION** (optional): Widget description
